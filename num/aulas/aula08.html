Aula 8: Solução de equações pelo método do ponto fixo.

Métodos iterativos

Os métodos de quebra, nos quais a aproximação da solução é progressivamente melhorada, são exemplos típicos de métodos iterativos. Esses métodos baseiam-se na repetição de um mesmo procedimento, que produz uma sequência de aproximações cada vez melhor, a partir de uma aproximação inicial.

Definição. Um método iterativo é um procedimento numérico no qual se constrói uma sequência de aproximações x1, x2, x3,… para a solução de um problema, a partir de uma aproximação inicial x0, por meio da aplicação repetida de uma mesma regra de cálculo. Dizemos que o processo iterativo converge se a sequência {xk} possui limite e esse limite coincide com a solução do problema x.
$\lim_{k\to\infty}x_k=x.$

Cada repetição do procedimento recebe o nome de iteração. Em cada iteração, a aproximação obtida na etapa anterior é utilizada para calcular uma nova aproximação, esperando-se que o erro diminua a cada passo. O processo iterativo deve ser interrompido quando a aproximação alcança a precisão desejada.

Na prática, o processo iterativo é interrompido após um número finito de iterações, quando um critério de parada é satisfeito, como, por exemplo, quando a diferença de duas aproximações (chamada o valor residual associado ao problema), torna-se menor do que uma tolerância ε previamente fixada,

∣ xk − xk−1 ∣ < ε.

Método do ponto fixo

Em muitos métodos iterativos, a sequência {xk} é definida a partir de uma função de iteração φ : D ⊂ ℝ → D, de modo que cada termo da sequência é obtido pela relação

xk = φ(xk−1) para k = 0, 1, 2, ….

Nesse contexto, o valor inicial x0 deve pertencer ao domínio D da função φ.

Se a sequência gerada por essa regra possui limite x e se a função φ é contínua, então o limite satisfaz a equação

x = φ(x).

Assim, a solução procurada pode ser caracterizada como um ponto fixo da interseção do gráfico da função de iteração y = φ(x) e a reta y = x.

A figura mostra que, neste caso, a sequência gerada pelo processo iterativo converge para o ponto fixo, sendo que cada iteração melhora a aproximação da solução exata x. Podemos ver que a trajetória das iterações permanece entre os gráficos y = φ(x) e y = x.

Quando a função de iteração é decrescente, a convergência também pode ocorrer. Nesse caso, as aproximações alternam os lados do ponto fixo, mas a distância até ele diminui a cada iteração.

Porém, o método de ponto fixo não garante a convergência: a sequencia x1, x2, x3,… obtida pela função φ(x) não necessariamente aproxima o valor de x.

Também pode haver divergência com uma função de iteração decrescente. Nesse caso, as aproximações continuam alternando os lados do ponto fixo, mas a distância até ele aumenta a cada iteração.

Nestes exemplos, mesmo que x0 esteja próximo a x, os valores x1, x2, x3,… podem ficar cada vez mais distantes do ponto x: x = φ(x). Uma condição suficiente da convergência pode ser formulada como um teorema seguinte:

Teorema sobre convergência do método do ponto fixo . Seja φ(x) uma função que possui a derivada contínua em um intervalo I centrado em x, tal que x=φ(x). Se
∀x ∈ I: |φ'(x)| ≤ M < 1,
então a seqüência definida por
xk = φ(xk−1),
para qualquer aproximação inicial ∀x0 ∈ I converge a x:
$\displaystyle{\lim_{k\to\infty}x_k=x.}$
Demonstração. O erro absoluto da aproximação na iteração k é
$\Delta x_k = \left|x_k − x\right| = \left|\phi(x_k) − \phi(x)\right|.$
Supondo que xk−1 ∈ I, como φ(x) possui a derivada contínua, por teorema do valor médio existe ξ entre xk−1 e x, tal que
$\Delta x_k = \left|\phi(x_k) − \phi(x)\right| = \left|\phi\prime(\xi)\right|\cdot\left|x_k − x\right| \leq M\cdot\left|x_k − x\right| = M\cdot\Delta x_{k−1},$
pois ξ ∈ I. Repetindo k vezes, obtemos
$\Delta x_k \leq M^k\cdot\Delta x_0,$
que vai para 0, pois M<1. Assim,
$\displaystyle{\lim_{k\to\infty}\Delta x_k = \lim_{k\to\infty}\left|x_k − x\right| = 0.}$ ∎

Na prática, se conseguimos provar que a função de iteração |φ'(x)| < 1 no intervalo suficientemente grande, então o algoritmo converge.

Estimativa do erro para o método de ponto fixo

Para estimar o erro da aproximação na iteração k, podemos comparar duas aproximações consecutivas:

$\Delta x_k = |x_k − x| = |x_k − x_{k+1} + x_{k+1} − x| \leq |x_k − x_{k+1}| + |x_{k+1} − x| = |x_k − x_{k+1}| + \Delta x_{k+1} \leq |x_k − x_{k+1}| + M\cdot\Delta x_k.$

Assim,

$\Delta x_k \leq \displaystyle{\frac{1}{1 − M}}\cdot|x_{k + 1} − x_k|.$

Finalmente,

$\Delta x_k \leq M\cdot\Delta x_{k−1} \leq \displaystyle{\frac{M}{1 − M}}\cdot|x_k − x_{k−1}|.$

Essa desigualdade é útil como critério de parada. Se queremos garantir erro absoluto menor que ε, basta continuar as iterações até que

$\displaystyle{\frac{M}{1 − M}}\cdot|x_k − x_{k−1}|<\varepsilon.$

Mesmo quando desconhecemos o valor de M, podemos usar o valor residual |xk − xk−1| como critério aproximado. Nesse caso, não obtemos uma estimativa rigorosa do erro, mas esperamos que o erro seja da mesma ordem de grandeza que o residual.

Resumo

  1. Métodos iterativos constroem uma sequência de aproximações a partir de uma aproximação inicial.
  2. No método do ponto fixo, a equação é transformada na forma x=φ(x), e as aproximações são obtidas pela repetição xk=φ(xk−1).
  3. A convergência do método do ponto fixo não é garantida para qualquer função de iteração nem para qualquer aproximação inicial.
  4. Uma condição suficiente de convergência é |φ'(x)|<1 no intervalo considerado.
  5. Quando a convergência é garantida, podemos estimar o erro usando a diferença entre duas aproximações consecutivas.
