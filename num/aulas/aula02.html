Aula 2: Base decimal e binária. Conversão entre bases.

Representação de números inteiros

A convenção usual para representar os números é a interpretação dos dígitos como fatores das potências de 10: o algarismo da última casa, sendo número de unidades, é multiplicado por 1 que é $10^0$, o algarismo da penúltima casa é multiplicador de $10^1$, e assim por diante. Cada casa decimal corresponde à potência de 10 que tem que ser multiplicada pelo algarismo nesta casa e a soma destes produtos dá o número escrito. Esta representação é chamada por isso a representação na base decimal. Por exemplo:

$231=2\times10^2+3\times10^1+1\times10^0.$

Para representação na base decimal usam-se dez dígitos de 0 até 9.

De mesma maneira podemos utilizar a base que é um inteiro β qualquer (β>1):

$231_{\beta}=2\times\beta^2+3\times\beta^1+1\times\beta^0.$

Para representação na base β precisamos β dígitos distintos: de 0 até β-1.

A base β=2 é chamada base binária. Somente dígito 0 e dígito 1 é usado para representar qualquer número:

$1101_2=1\times2^3+1\times2^2+0\times2^1+1\times2^0=8+4+0+1=13.$

Conversão da base decimal para a base binária

Enquanto para converter um número da base qualquer para a base decimal podemos utilizar a definição, como nós acostumamos fazer contas na base decimal, a conversão para outra base usa a observação seguinte: o dígito da última casa na base β é sempre o resto da divisão do número por β. Isto é simplesmente a consequência do fato que todos os outros dígitos são multiplicados por potências positivas de β, não contribuindo no resto da divisão.

Assim, dividimos o número por β e o resto é o dígito da última casa. O resultado da divisão dá o número que na base β é construído com todos os dígitos a menos o último. Isto é porque o resultado da divisão por β diminui todas as potências de β por 1. Repetindo a divisão obtemos os dígitos de todas as casas na base β, começando do último e acabando com o primeiro.

Fração decimal

A mesma convenção é usada para representar os números fracionais. Na fração decimal, depois do ponto que separa separa a parte inteira da parte decimal, encontram se as casas que correspondem às potências negativas de dez:

$30.21=3\times10^1+0\times10^0+2\times10^{-1}+1\times10^{-2}.$

A parte fracionária pode ser finita ou infinita.

A parte fracional na base β é representada pela combinação das potências negativas de β:

$30.21_{\beta}=3\times\beta^1+0\times\beta^0+2\times\beta^{-1}+1\times\beta^{-2}.$

Utilizando a base 2, obtemos uma fração binária

$10.101_2=1\times2^1+0\times2^0+1\times2^{-1}+0\times2^{-2}+1\times2^{-3}=2+0.5+0.125=2.625.$

Como qualquer potência negativa de 2 é representada por fração decimal finita, podemos observar que qualquer fração finita na base binária será finita na base decimal.

Conversão de uma fração para a base binária

Para converter uma fração para a base β observamos que, se multiplicarmos a parte fracional por β, a parte inteira do resultado é o primeiro algarismo depois do ponto na base β. A parte fracionária do resultado vai ter o resto das casas na base β. Podemos repetir a multiplicação para determinar tantas casas fracionárias quantas forem necessárias.

Para obter a representação na forma de uma fração binária multiplicamos repetidamente a parte fracional por 2. Por exemplo:

$0.65\times2=1.3$
$0.3\times2=0.6$
$0.6\times2=1.2$
$0.2\times2=0.4$
$0.4\times2=0.8$
$0.8\times2=1.6$
$0.6\times2=1.2$
$\ldots$

Então

$2.65=10.1010011\ldots_2$

Neste exemplo a fração decimal finita é a infinita na base binária.

Resumo

  1. Existem apenas 10 tipos de pessoas: aqueles que sabem representar qualquer número na base binária, e aqueles que não sabem.
  2. A representação na base binária utiliza somente dois dígitos distintos: 0 e 1.
  3. A fração finita na base decimal pode ser infinita na base binária, embora que o contrário é impossível: qualquer fração binária finita pode ser representada pela fração decimal finita.
